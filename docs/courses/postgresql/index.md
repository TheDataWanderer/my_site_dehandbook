# PostgreSQL для аналитиков

Практический курс по PostgreSQL для работы с данными в аналитических задачах.

Курс основан на реальном опыте работы с базами данных и предназначен для тех, кто хочет уверенно использовать PostgreSQL в повседневной работе с данными.

---

## Для кого этот курс

- аналитики и специалисты, работающие с данными
- сотрудники, которым необходимо работать с базами данных

---

## Что входит в курс

### 📘 Лекции
В разделе **Лекции**:
- основы работы с PostgreSQL
- продвинутые техники запросов
- оптимизация и администрирование
- примеры из реальных задач

### 📝 Домашние задания
В разделе **Домашние задания**:
- практические задания для закрепления материала
- контрольные и экзаменационные работы
- разборы решений

### 🔗 Дополнительные материалы
В разделе **Ресурсы**:
- описание учебной базы данных
- полезные ссылки и ресурсы
- инструкции по настройке

---

## Как работать с курсом

1. Изучайте лекции по порядку  
2. Закрепляйте материал с помощью домашних заданий  
3. Используйте дополнительные материалы как справочник  

Курс можно проходить в удобном темпе, возвращаясь к нужным темам по мере необходимости.

---

## Требования

Перед началом работы убедитесь, что у вас установлена среда для выполнения запросов (DBeaver, DataGrip, pgAdmin, psql или иной инструмент).

В курсе используется демо база от Postgrespro  
[Демо база](https://postgrespro.ru/education/demodb)
