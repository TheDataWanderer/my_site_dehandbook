# Excel для менеджеров

Авторский практический курс по Excel для работы с данными в управленческих и аналитических задачах.

Курс основан на реальном опыте работы с отчётами, таблицами и бизнес-метриками и предназначен для тех, кто хочет уверенно использовать Excel в повседневной работе.

---

## Для кого этот курс

- менеджеры и руководители
- аналитики и специалисты, работающие с отчётами
- сотрудники, которым регулярно приходится обрабатывать данные в Excel

---

## Что входит в курс

### 📘 Лекции
В разделе **Лекции**:
- базовые и продвинутые приёмы работы с данными
- примеры из реальных задач
- рекомендации по оптимизации работы в Excel

### 📝 Домашние задания
В разделе **Домашние задания**:
- практические задания для закрепления материала
- задачи, приближенные к реальным рабочим кейсам

### 🔗 Дополнительные материалы
В разделе **Ресурсы**:
- полезные статьи и ссылки
- шпаргалки и сочетания клавиш
- дополнительные материалы для углубления знаний

---

## Как работать с курсом

1. Изучайте лекции по порядку  
2. Закрепляйте материал с помощью домашних заданий  
3. Используйте дополнительные материалы как справочник  

Курс можно проходить в удобном темпе, возвращаясь к нужным темам по мере необходимости.
